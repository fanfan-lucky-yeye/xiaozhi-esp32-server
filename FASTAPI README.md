# 语音问答服务部署指南

## 📋 项目概述

本项目是一个基于 FastAPI 的语音问答服务，支持文本/语音输入，返回文本/语音输出。

### 技术栈

| 组件 | 技术 | 版本 |
|------|------|------|
| 框架 | FastAPI | 0.115.0 |
| 服务器 | Uvicorn | 0.31.1 |
| ASR | FunASR | 1.2.3 |
| TTS | EdgeTTS | 7.0.0 |
| LLM | ChatGLM | 4.0 |
